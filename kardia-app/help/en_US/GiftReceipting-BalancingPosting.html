Balancing: Print List of Receipted Checks

Start with this Receipted Checks report when you're ready to start balancing a batch.


  1. Close out of the window where you were entering receipts.

  2. With the correct batch selected, click Report: Receipted Checks



  1. Click the Print button



  1. Click OK

  2. Get the report from the printer and check it against the adding machine tape/manual tracking.

  3. If you need to make corrections to the batch, close the report, click the Enter Receipts... button and make any changes. Then run the report again.




    Posting a Receipts Batch

After everything balances...


  1. With the correct batch selected, click Post Batch to CR Journal



  1. Check for the Post check mark to make sure it posted.





    Printing the Final Donation Detail Report

Print this report after everything balances and you've posted the batch.


  1. With the correct batch selected, click Report: Donation Detail



  1. Click the Print button



  1. Click Properties and make sure the printer settings are set correctly, especially if you are wanting to print the report double-sided (duplex). Click OK to close the Properties box.

  2. Click OK to print to the printer.
