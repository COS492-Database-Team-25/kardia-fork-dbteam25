  1. Start Kardia from the icon on the desktop

  2. Enter your user name and password in the Authentication box. Click OK.

  3. Click on the Gift tab

  4. Click on Receipts Batches

  5. Make sure Select Period displays the correct month.

  6. Click the New Batch button

  7. Enter Receipts mm/dd/yyyy in the Description box.

  8. Click Save





Enter Receipts


  1. Click Enter Receipts for #


  1. Click New Receipt


  1. Search to see if the donor is in the system

Different ways you can do this:

      1. Type in the partner ID and press Tab or

      2. Type in any part of the donor name (Bergen Jo) and press F3. Use the keyboard arrows to find the correct donor name and press Tab or

      3. Type in zip code and part of the donor name (30277 Bergen or 30277 Joe) and press F3. Use the keyboard arrows to find the correct donor name and press Tab

        Tip: Think about the most unique part of their name or zip. If the person is name Danica Smith, search for Danica, not Smith. There will probably be fewer of those in the database.


  1. If the search returned multiple entries and you're not sure which is right, click on the one that might be correct and look at the address/giving history in the donor window.


  2. If the donor is not the one you are wanting, click in the Donor number field and search again.
    If a donor is not found in Kardia or if there is an address change needed, set the gift aside to search for/enter later in FilePro and move on to the next gift.


  1. If the right Donor is selected, use Enter to jump to the designation.



In the Designation section

  1. Type in check number Press Tab

  2. Type in amount Press Tab

  3. Press Tab two more times to get past Personal Gift and Anonymous Gift

  4. Designation code: Type in the Designation code OR search for it
    Searching for designation code:

    1. Type part of the designation description in the designation field. Part of the description might include missionary last name or part of the project name.

    2. Use the arrow keys to highlight the correct designation.

    3. Press tab and glance at Designation code description to make sure it's right.

  5. Press Enter if no comments

    or if there are comments, press tab to get to comments, enter the comments, and press Enter

  6. Write the receipt number on the envelope


Another designation for the same receipt?: The cursor should be back in the check number box and you can immediately start entering the next designation for that receipt. Repeat steps 15-19.


Ready for the next receipt? Click New Receipt or press Enter again.


Not wanting to start a new receipt or a new designation? Press Esc after saving the last designation.
