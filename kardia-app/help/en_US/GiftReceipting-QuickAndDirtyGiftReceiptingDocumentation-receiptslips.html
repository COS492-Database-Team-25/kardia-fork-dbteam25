  1. Start Kardia from the icon on the desktop

  2. Enter your user name and password in the Authentication box. Click OK.

  3. Click on the Gift tab

  4. Click on Receipts Batches

  5. Make sure Select Period displays the correct month.

  6. Click the New Batch button

  7. Enter Receipts mm/dd/yyyy in the Description box.

  8. Click Save

Enter Receipts

  1. Click Enter Receipts for #


  1. Click New Receipt


  1. Enter the Donor number

  2. Press Tab

  3. Donor right? Press Enter to jump to designation


In the Designation section

  1. Type in check number Press Tab

  2. Type in amount Press Tab

  3. Press Tab two more times to get past Personal Gift and Anonymous Gift (but remember to check these boxes if the gift is a personal gift or anonymous gift!)

  4. Type in the Designation code.

  5. Press tab and glance at Designation code description to make sure it's right.

  6. Press Enter if no comments

    or if there are comments, press tab to get to comments, enter the comments, and press Enter

  7. Write the receipt number on the envelope


Another designation for the same receipt?: The cursor should be back in the check number box and you can immediately start entering the next designation for that receipt. Repeat steps 14-19.


Ready for the next receipt? Click New Receipt or press Enter again.


Not wanting to start a new receipt or a new designation? Press Esc after saving the last designation.
